## 改善案

以下のような構成で、より詳細なREADMEを提供することを推奨します：

```markdown
# RNGNewAuraNotifier

The application notifies Windows Toast and Discord Webhook of Aura acquired in [Elite's RNG Land](https://vrchat.com/home/world/wrld_50a4de63-927a-4d7e-b322-13d715176ef1).

## 機能

- VRChatのログファイルを監視して新しいAura獲得を自動検出
- Windows Toast通知でデスクトップに通知
- Discord Webhook経由でDiscordサーバーに通知
- 自動アップデート機能
- カスタマイズ可能な設定

## インストール方法

1. [最新リリース](https://github.com/tomacheese/RNGNewAuraNotifier/releases/latest)からアプリケーションをダウンロード
2. ダウンロードしたZIPファイルを任意の場所に展開
3. `RNGNewAuraNotifier.exe`を実行

## 設定方法

1. システムトレイのアイコンを右クリックし、「Settings」を選択
2. VRChatのログフォルダを設定（デフォルト: `%USERPROFILE%\AppData\LocalLow\VRChat\VRChat`）
3. Discord通知を利用する場合は、Discord WebhookのURLを設定

## システム要件

- Windows 10 バージョン 1809 (ビルド 10.0.17763.0) 以上
- .NET 9.0 ランタイム
```
