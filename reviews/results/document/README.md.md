# RNGNewAuraNotifier

VRChatの「[Elite's RNG Land](https://vrchat.com/home/world/wrld_50a4de63-927a-4d7e-b322-13d715176ef1)」ワールドで獲得したオーラをWindows Toast通知とDiscord Webhookで通知するWindowsアプリケーションです。

![スクリーンショット](docs/images/screenshot.png)

## 特徴

- リアルタイムでVRChatログを監視し、新しいオーラ獲得を検出
- Windows Toast通知で即座に獲得通知
- Discord Webhookを使用したサーバーへの通知連携
- システムトレイに常駐し、軽量で使いやすいインターフェース
- 全オーラのデータベースによる正確な情報表示

## インストール

### 必要条件

- Windows 10 バージョン1809以降
- .NET 9.0 ランタイム
- VRChat

### ダウンロードとインストール

1. [リリースページ](https://github.com/username/RNGNewAuraNotifier/releases)から最新バージョンをダウンロード
2. ダウンロードしたファイルを実行し、画面の指示に従ってインストール
3. 初回起動時に設定ウィザードが表示されます

## 使い方

1. VRChatのログディレクトリを設定（通常は自動検出されます）
2. Discord通知を使用する場合は、Webhook URLを設定
3. アプリケーションをバックグラウンドで実行したままVRChatを起動
4. Elite's RNG Landでオーラを獲得すると自動的に通知されます

詳細な設定方法や高度な使い方については、[ユーザーガイド](docs/user-guide.md)を参照してください。
